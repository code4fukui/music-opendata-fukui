# music-opendata-fukui

このプロジェクトは、福井の観光体験をオープンデータで高めるシンプルな音楽プレーヤーを提供しています。

## 機能
- 福井の観光に関するオープンデータを活用した音楽トラックを再生できます
- 現在のトラックのアルバムアート、トラック情報、歌詞を動的に表示します
- プレイリストの曲を自動的に次々と再生します
- プレイリストからトラックを閲覧、選択できます

## 必要環境
Deno環境が必要です。

## 使い方
以下のコマンドでプレイリストをダウンロードできます:

```sh
deno -A https://code4fukui.github.io/music-opendata-fukui/download.js [playlist id]
```

`index.html`ファイルをWebブラウザで開くと音楽プレーヤーを使えます。

## データ・API
このプロジェクトでは福井のオープンデータを活用しています。
