# music-opendata-fukui

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

The `music-opendata-fukui` project provides a simple music player that showcases how open data can be used to enhance the tourism experience in Fukui, Japan.

## Features
- Plays a collection of music tracks about Fukui's tourism powered by open data
- Dynamically displays album artwork, track information, and lyrics based on the current track
- Automatically transitions to the next track in the playlist
- Allows users to browse and select tracks from the playlist

## Usage

To download the playlist, run the following command:

```sh
deno -A https://code4fukui.github.io/music-opendata-fukui/download.js [playlist id]
```

To run the music player, open the `index.html` file in a web browser.

## Data / API
This project utilizes open data from various sources in Fukui to provide a rich tourism experience. The data includes information about local landmarks, events, and transportation.
